Use Cases

  • Ensure the card text is concise and descriptive.
  • Ensure the hover: state contrasts with the default color of the button.
  • Use the horizontal card for the desktop and tablet view and the vertical for mobile.

Accessibility Considerations

The card text should be accessible using the keyboard to allow screen reader usage. Ensure that the WCAG construct ratio of the card-body is 6.39:1 by keeping the text #FFFFFF and the background color #000000.
