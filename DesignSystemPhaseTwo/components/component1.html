Use Cases

  • Ensure that the navigation bar is on the top of each page.
  • Ensure that the page link where the user is currently is in bold with the primary color.

Accessibility Considerations

The navigation bar should be accessible using the keyboard to allow screen reader usage. The labels must be short and descriptive to ensure easy navigation. Keep the navigation bar background color #FFFFFF and the text color #000000 to ensure that the WCAG construct ratio between the background and text is 6.39:1.
